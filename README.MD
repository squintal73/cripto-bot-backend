### Endpoints

 Path | Method | Description
 ---|---|---
 /Lista           | GET    | Lista Moedas
 /Cadastra        | POST   | Cadastra Moedas
 /Remove          | PUT    | Atualiza Moedas
 /Detalha/id      | DELETE | Remove Moedas
 /Detalha/id      | GET    | Detalha Moedas
